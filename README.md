# CRM-Application-For-Jewel-Management
This project is developed as part of the Nan Mudhalvan Salesforce Project to implement real-world CRM functionalities for a Jewel Management System.

Features: 1.Manage jewellery stock, billing, and customer details easily. 
          2.Generate automatic invoices with tax and discount options.
          3.Track sales, suppliers, and inventory in real-time.
          4.Send alerts for payments and offers via email/SMS.
          5.Secure access with role-based user control

Technologies Used: 1.HTML, CSS, and JavaScript for front-end design.
                   2.Salesforce CRM platform for application development.
                   3.Apex and SOQL for backend logic and database operations.
                   4.Reports and Dashboards for analytics and data visualization.
